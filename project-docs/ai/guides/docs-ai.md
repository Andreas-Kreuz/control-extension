# AI Documentation Guide

## When to Update

When central concepts, contract terms, or project conventions change, update the AI doc tree under `project-docs/ai/`.

## Tree Rules

See [../principles/ai-docs.md](../principles/ai-docs.md) for the structural rules:

- Tree structure, downward-only links, one parent per document
- Every agent starts at `agents.md`, follows one clear path downward
- Cross-topic context goes in the nearest common parent, not sideways
