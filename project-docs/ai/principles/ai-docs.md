# CE Principle: AI Docs

AI entry point: `agents.md`

Rules:

- Keep `project-docs/agents.md` and `project-docs/ai/**/*.md` compact, high-signal, and easy to scan for AI agents.
- Prefer short sections, stable terminology, and minimal repetition.
- Use only downward-facing links to more specific documents.
- Keep the link structure as a tree, not a graph.
- Each AI document must be reachable from exactly one link in one document above it.
- Put cross-topic context into the nearest common parent instead of linking sideways or upward.

Rule: every agent starts at `agents.md`, then follows one clear path downward.
