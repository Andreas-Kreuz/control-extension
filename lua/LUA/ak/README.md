# Diese Dateien sind nur für das Bereitstellen der alten API gedacht

Die Dateien unter `ak/` enthalten den Hinweis `-- THIS IS A COMPAT FILE FOR THE OLD API.`, da sie nur für die Verwendung der alten API gedacht sind, wenn alte Anlagen geöffnet werden.
Bitte diese Dateien nur verwenden, wenn noch nicht auf die Lua-Dateien in `ce/` umgestellt wurde.

Hier findest du mehr Informationen zum [Kompatibilitätslayer](../ce/Using_ak_compat_layer.md)
